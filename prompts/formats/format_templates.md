# Format Templates

These are the available post formats. The generator selects the format that best serves the extracted point. Format and hook must work together — not every hook suits every format.

---

## Format 1: The Single Take
**Best for:** Bold opinions, blunt facts, direct observations
**Structure:** 1–4 sentences. One idea, stated clearly and completely. No thread. No list.
**Character target:** 80–240 characters
**Hook compatibility:** Blunt Fact, Contrarian Opening, Direct Opinion
**Example shape:**
```
[Hook sentence that states the whole idea]
[Optional: one sentence that deepens or complicates it]
[Optional: one sentence that lands the implication]
```

---

## Format 2: The Mini-Thread (3–5 tweets)
**Best for:** Breakdowns, how-something-works explanations, contrarian positions with evidence
**Structure:** Tweet 1 = hook (must stand alone). Tweets 2–4 = development. Tweet 5 = landing point or CTA.
**Character target per tweet:** 200–280
**Hook compatibility:** All hook types
**Thread format:**
```
Tweet 1: [Hook — the whole post in one line, creates tension or curiosity]
Tweet 2: [The setup — what most people think/do]
Tweet 3: [The turn — what's actually true/what happened]
Tweet 4: [The specific mechanism or evidence]
Tweet 5: [The landing — implication, takeaway, or soft CTA]
```

---

## Format 3: The Breakdown Thread (6–8 tweets)
**Best for:** Technical deep-dives, reverse engineering stories, step-by-step cracking narratives
**Structure:** Tweet 1 = hook with a specific promise. Middle tweets = the actual breakdown. Final tweet = summary + what they can do with this.
**Character target per tweet:** 200–280
**Hook compatibility:** Specific Failure, Curiosity Gap, Before/After
**Thread format:**
```
Tweet 1: [Hook + explicit promise: "Here's how it actually works:"]
Tweet 2–6: [Step-by-step or layer-by-layer breakdown. Each tweet = one layer]
Tweet 7: [The implication — what this means for how you build/think]
Tweet 8 (optional): [CTA or call for discussion]
```

---

## Format 4: The Day Series Post
**Best for:** Build-in-public progress updates, milestone posts, honest check-ins
**Structure:** Opens with "Day X" anchor. One honest status line. One specific thing that happened (progress or setback). Optional: what's next.
**Character target:** 150–280 for single post, or 3–4 tweet thread
**Hook compatibility:** Day Series Anchor only
**Example shape:**
```
Day [X] of [cracking challenge] to [specific goal].

[One sentence: honest current status]
[One sentence: specific thing that happened — concrete detail]
[Optional: what happens next]
```

---

## Format 5: The Bold Take + Evidence
**Best for:** Contrarian positions, industry critiques, "here's why everyone's wrong about this"
**Structure:** State the take cold. Spend 2–3 tweets on the evidence or reasoning. Land with the implication.
**Character target:** 3–5 tweet thread
**Hook compatibility:** Contrarian Opening, Direct Opinion
**Thread format:**
```
Tweet 1: [The take — stated plainly, no hedging]
Tweet 2: [Evidence or example 1 — specific]
Tweet 3: [Evidence or example 2 — specific, or the mechanism that explains it]
Tweet 4: [Landing — what this means for how you should act]
```

---

## Format 6: The Specific Observation (Single)
**Best for:** Pattern observations, "I've noticed that..." content, behavior analysis
**Structure:** Name the pattern specifically. One brief expansion. Done.
**Character target:** 100–240
**Hook compatibility:** Specific Observation, Curiosity Gap
**Example shape:**
```
[Name the pattern — specific enough to be recognizable]
[One sentence: why it happens or what it implies]
```

---

## Format Selection Guide

| Extracted Point Type | Recommended Format |
|---|---|
| A counterintuitive fact | Single Take or Bold Take + Evidence |
| A technical mechanism | Breakdown Thread |
| A personal failure/lesson | Mini-Thread or Single Take (Specific Failure hook) |
| A build milestone | Day Series Post |
| A contrarian position | Bold Take + Evidence |
| A pattern observation | Specific Observation or Mini-Thread |
| A data point | Single Take (Blunt Fact hook) |
